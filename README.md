# ApexPlanet Task 4 - Data Storytelling & Validation

## 📊 Project Overview
This project focuses on transforming sales data into meaningful insights using data storytelling and basic statistical validation.

## 🎯 Objective
To analyze sales data and identify key business insights to support decision-making.

## 📂 Dataset
The dataset used is cleaned_sales.csv containing:
- Order ID
- Category
- Product
- City
- Payment Mode
- Revenue

## 📈 Key Insights
- Electronics generates the highest revenue
- Hyderabad has the highest sales
- UPI is the most preferred payment method

## 🧪 Hypothesis
UPI is the most preferred payment method among customers.

## ✅ Validation
The analysis shows that UPI transactions are highest, supporting the hypothesis.

- Excel
